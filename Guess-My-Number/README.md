# Guess My Number

The objective of this project is to guess a number between 1-20.
When you guess the number, the score will be recorder in the HighScore.
If in the next game you guess the number in less attempts, the HighScore will be updated.
Everytime you fail to guess the number the score that starts in 20 will decrease until you guess. You lose the game if you reach the score 0.
You can start a new game - Click in the Button Again.

Every time you want to check if the number is the correct one - Click in the Button Check - we can have the following outputs:

- [Correct Number](#correct-number)
- [Incorrect Number](#incorrect-number)
  - [Input is not a number](#input-is-not-a-number)
  - [Input is too high](#input-is-too-high)
  - [Input is too low](#input-is-too-low)
